This project is a full-stack replica of the Udyam registration form, built using React for the frontend and Express + MongoDB for the backend.
It features:
- ✅ Real-time validation for Aadhaar, PAN, email, and other fields
- ✅ Modular UI components with responsive design (Tailwind CSS)
- ✅ Backend form submission handling with server-side validation
- ✅ MongoDB persistence via Mongoose schema
